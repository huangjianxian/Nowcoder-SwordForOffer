## 第十一题 二进制中1的个数

### 题目描述

> 输入一个整数，输出该数32位二进制表示中1的个数。其中负数用补码表示。

### 法一：十进制转换为二进制

**思路：** 

> 由于输入的是十进制的整数，可以先将十进制转换为二进制，再记录二进制中为1的个数即可。   
十进制转换为二进制的方法：十进制除以二，每一步除以二如果有余数，将余数记录下来。直到除到结果为0，然后再将余数按照反方向写出即可。   

**待解决问题：** 

> 负数的表示。


**参考别人的代码：** 

```C#
        public int NumberOf1(int n)
        {
            int count = 0;
            int flag = 1;
            while (flag != 0)
            {
                if ((n & flag) != 0)
                {
                    count++;
                }
                flag = flag << 1;
            }
            return count;
        }
```
